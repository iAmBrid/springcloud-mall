# springcloud-mall 畅购商城项目

## 一. 项目结构
```$xslt
springcloud-mall
├─changgou-parent    父工程
├─changgou-auth      oauth2.0授权认证管理中心
├─changgou-common    
├─changgou-common_db
├─changgou-eureka
├─changgou-gateway   微服务网关
├─changgou-reverser
├─changgou-service   微服务
├─changgou-service_api  微服务pojo、feign、hystrix熔断限流配置
├─changgou-transaction_fescar   分布式事务fescar
└─changgou-web                  web层
```
